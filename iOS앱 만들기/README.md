## iOS 앱 만들기는 "비판적 클론코딩"을 토대로 진행했습니다.

### 비판적 클론코딩이란 ?
단순히 코드를 따라서 치는 것보다는, 능동적으로 생각하며 코드를 작성하는 것이 더 좋다고 생각합니다.   
다른 사람에게서 좋은 코드를 흡수하지만, 항상 "왜" 라는 생각을 가지며 코드를 작성합니다.   
"나는 이렇게도 생각해보았습니다." 라는 스탠스를 취하며 저만의 생각이 추가적으로 담긴 앱을 만들었습니다.
